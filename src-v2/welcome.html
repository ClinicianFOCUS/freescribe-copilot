Welcome to FreeScribe

FreeScribe is a powerful browser extension that helps you transcribe audio, generate notes, and manage your medical documentation effortlessly.

Key Features:

  • Flexible Processing Options

    Choose between local or server-based transcription and LLM processing to suit your needs

  • Real-time Audio Transcription

    Convert speech to text instantly with high accuracy and automatic punctuation

  • AI-powered Note Generation

    Transform raw transcriptions into structured medical notes using advanced AI processing

  • Customizable Templates

    Use and modify pre-built templates or create your own custom prompts for different medical scenarios

  • Pre/Post Processing

    Apply custom rules and formatting to your notes before and after AI processing

  • History Management

    Track, search, and manage your transcription and note history with easy access to past records

  • Device Flexibility

    Support for multiple audio input devices and microphone configurations

  • Security First

    Process all data locally with optional cloud features, ensuring complete control over sensitive information

  • Easy Export

    Copy your notes in just one click

Keyboard Shortcuts:

  • Open FreeScribe Extension: Alt + Shift + F
  • Start/Stop Recording: Alt + Shift + R
  • Configure: Alt + Shift + Comma (,)

To get started, click on the FreeScribe icon in your browser toolbar. You can record audio, transcribe it, generate notes, and manage your history all from one convenient interface.

For the best experience, we recommend configuring FreeScribe to work with your preferred settings here.

Installations (Optional):

If using server-based processing, you will need to install the following dependencies:

  • Local LLM Container: Required for running the local language model. Follow the installation instructions in the repository.
  • Speech to Text Container: Needed for audio transcription. Please refer to the repository for installation details.
  • FreeScribe: The main project repository. Please follow the installation instructions provided in the repository.

For detailed installation steps, please refer to the README files in each of the above repositories.
